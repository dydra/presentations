NXP : the Data Hub : what is the technology?

Dydra : a cloud-based RDF database and SPARQL processor, delivered as a service (DaaS)

  • In development since 2009, incorporated in 2011. Technology team based in Berlin. GmbH since 2013.
  • Provides free hosting for ontologies and small Linked Data datasets (“GitHub for RDF data”), each repository is a SPARQL endpoint.
  • Commercial arrangements available for managed turnkey hosting of larger datasets on dedicated hardware in the cloud or on-site, with SLAs.

NXP : the Data Hub : development support: repositories

Features:

  • Standards support: SPARQL1.1 Query, Update, Federated Query, Graph Store HTTP Protocol
  • First-class revision history, query access to revisions, temporal federation across revisions.
  • Materialized views and strong cache identifiers, efficient remote access to named result sets
  • SPARQL extensions—implicit reification, collation, temporal operators, provenance operators, etc.
  • All major RDF serialization formats, and others:
    • JSON-LD
    • Turtle
    • RDF/XML
    • RDF/JSON
    • N-Triples/ N-Quads
    • TriX, Pajek
    • Graphviz
    • etc.
